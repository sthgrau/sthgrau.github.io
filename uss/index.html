This is a sandbox environment for user-scripts. Sometimes, you want to test some things out and don't have a page for it.. You can use this instead..

To create a tab inside a user-script, just run the following

var par = createNewTab('dhtmlgoodies_tabView2','<Your Title>',"[OPTIONAL: Your Initial Text]",'[OPTIONAL: a url to load into the tab via ajax]',(true or false, to put a close button or not);

For example:

var par = createNewTab('dhtmlgoodies_tabView2','<Your Title>',"Testing a thing",null,true);

jQuery is preloaded
